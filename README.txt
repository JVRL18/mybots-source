This is a MYBOTS APP BASE SOURCE V3,

Here we have a ready to go bot that is Mongo, Node, DJS Stack

Snippets are recommended to install as well to boost production

You should note also BY *DEFAULT* interaction create is just enabled for guild ADMINS

Working with buttons you'll be using the commands_fixed file at src/

DEFAULT COMMANDS

- Default commands should have the client ID inserted into superAdmin array at config.json in order to let client change sensitive data
